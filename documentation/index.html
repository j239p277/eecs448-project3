sendPuck

Sends the puck location and velocities to the server.

sendPuck(px: float, py: float, pz: float, vxl: float, vyl: float, vzl: float, vxa: float, vya: float, vza: float)
Parameters
px (float) x coordinate of puck's position
py (float) y coordinate of puck's position
pz (float) z coordinate of puck's position
vxl (float) linear velocity in the x direction
vyl (float) linear velocity in the y direction
vzl (float) linear velocity in the z direction
vxa (float) angular velocity in the x direction (roll)
vya (float) angular velocity in the y direction (pitch)
vza (float) angular velocity in the z direction (yaw)

default_color (array) array representing default color info in rgb format
scene (scene) scene object containing all gameplay elements
